# 🚀 LeetCode Progress Tracker

A responsive web application that allows users to search for any LeetCode profile and view their coding statistics in a clean and interactive dashboard.

## 🌐 Live Demo

https://sumit-parshadgit.github.io/sumit_parshad.github.io/leetcode-progress-tracker/



---

## ✨ Features

- 🔍 Search any public LeetCode username
- 📊 Visual progress circles for Easy, Medium and Hard problems
- 📈 Overall statistics dashboard
- 📋 Displays:
  - Easy solved & submissions
  - Medium solved & submissions
  - Hard solved & submissions
  - Total solved problems
  - Total submissions
  - Total questions available
- ⚠️ Handles invalid usernames gracefully
- 📱 Responsive and modern UI

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- LeetCode Stats API

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/leetcode-progress-tracker.git
```

Open the project folder

```bash
cd leetcode-progress-tracker
```

Run the project

Simply open `index.html` in your browser.
